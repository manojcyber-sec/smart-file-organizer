# smart-file-organizer
Python automation tool to organize files by type Public

## What this does
Automatically sorts files into folders based on file type such as Images, Documents, Code, Music, Videos, and Archives.

## Why I built this
To practice Python file handling and automation concepts.

## Tools Used
Python, os module, shutil module

## How to Run
1. Create a folder named test_folder
2. Put mixed files inside it
3. Run:
   python organizer.py

## Features
- Auto folder creation
- Moves files by extension
- Handles unknown files into Others folder
